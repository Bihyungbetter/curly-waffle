Student Copyright Checklist

This checklist documents our adherence to copyright laws and proper attribution of resources used.

Images and Graphics

All images used on the website are either original creations, licensed through appropriate channels, or sourced from royalty-free image libraries. Each image is properly credited in the site's source code.

Fonts and Typography

All fonts used are sourced from Google Fonts, which are free for commercial use. Font Awesome icons are used under their free license terms.

Content and Copy

All written content is original and created specifically for this project. Menu items and descriptions are fictional and created by our team.

Code and Libraries

The website uses AOS (Animate On Scroll) library, which is open-source and used in accordance with its MIT license. All custom code was written by our team members.

Design Elements

UI design patterns and layouts are inspired by industry standards but implemented uniquely for this project. No templates or pre-made designs were used.

Declaration: We hereby confirm that all content, code, images, and other resources used in this website adhere to copyright laws and are properly attributed where required.
